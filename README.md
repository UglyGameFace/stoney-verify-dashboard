# Stoney Verify Dashboard V3.8

Production-oriented full codebase with:

- Discord OAuth staff authentication
- realtime dashboard and ticket updates
- ticket categories
- transcript export
- live Discord member search
- role hierarchy viewer
- moderation helpers
- safer partial serverless role sync
- bot auto-sync worker
- env validation
- migration script
- Vercel deployment notes

## Setup

```bash
npm install
npm run check-env
npm run migrate
npm run build
npm run dev
```

Run the bot separately:

```bash
npm run bot
```
